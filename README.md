## 🚀 拡張機能名（仮称）：クイックメモ📝

### 🌟 概要

この拡張機能は、Chromeの**サイドパネル**を利用して、**プレーンテキスト**の一時的な書き込みスペースを提供します。

複数のブラウザタブをまたいで**テキストをコピー＆ペースト**する際や、ちょっとしたメモを**一時的に保管**したい場合に便利です。

> 💡 **ポイント:** メモを書いても、サイドパネルを閉じたり、Chromeを終了したりすると内容は**自動的にクリア**され、記録は残りません。**純粋な一時保管**に特化しています。

---

### 📝 使い方

1.  **サイドパネルを開く:** Chromeのツールバーにある**拡張機能のアイコン**をクリックします。
    * （アイコンが表示されない場合は、パズルのピースのアイコンから本拡張機能を見つけ、ピン留めしてください。）
2.  **テキストを書き込む:** 表示されたサイドパネル内のスペースに、自由にテキストを入力したり、コピーしたテキストを貼り付けたりしてください。
3.  **一時保管として活用:** 別のタブで作業する際も、このサイドパネルは開いたまま維持できるため、**タブを切り替えても**すぐにテキストを参照・利用できます。

---

### ⚠️ 注意事項

* **永続的な保存機能はありません。**
    * サイドパネルを閉じると、書き込んだ内容は**すべて消去**されます。大事な情報は必ず別の場所（メモ帳、ドキュメントなど）に保存してください。
    * Chromeの再起動時にも内容はクリアされます。

---

### 🛠️ 開発者情報

この拡張機能は、**Gemini**（Googleによってトレーニングされた大規模言語モデル）のサポートを受けて作成されました。
